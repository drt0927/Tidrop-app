# Tidrop

Tidrop은 Windows 환경에서 다운로드된 파일을  
자동으로 분류하고 정리해주는 데스크톱 애플리케이션입니다.

사용자는 아무 설정을 하지 않아도 되며,  
앱은 백그라운드에서 조용히 동작합니다.

---

## ✨ 주요 특징

- 📂 다운로드 파일 자동 분류
- ⚡ 파일 다운로드 직후 즉시 처리
- 🖥️ Windows 트레이에서 백그라운드 실행
- 🔒 개인정보 수집 없음
- ☁️ 서버 / 클라우드 통신 없음
- 📴 인터넷 연결 불필요 (완전 로컬 동작)

---

## 🔐 개인정보 및 보안

Tidrop은 사용자의 개인정보를 수집하거나 저장하지 않습니다.

모든 파일 처리 및 분류 작업은  
사용자의 PC 내부에서만 수행되며,  
어떠한 데이터도 외부로 전송되지 않습니다.

👉 자세한 내용은  
[개인정보 처리방침](privacy.html)을 참고해 주세요.
